# Game Library

An open-source game library management web application developed collaboratively using GitHub.

## Project Goal
The goal of this project is to create a simple dynamic website where users can browse and discover games through a web interface.

## Planned Features
- Display game titles
- Display cover images
- Short game descriptions
- Search/filter games
- Store game information in a database

## Technologies
- PHP
- MariaDB
- HTML/CSS
- JavaScript
- GitHub for collaboration

## Environment
The project is planned to run on a Raspberry Pi Zero 2 W using a Linux-based environment.
